Старт
1) грузим страницу
2) открываем start
3) грузим файл настройки
4) сканируем на странице
- поля/таблицы ввода/вывода данных
- блоки/кнопки (для скрытия/показа)
5) по данным из локального хранилища настраиваем окно абонента
6) включаем слушатели действий абонента

управление окнами
1) клик по активному элементу
2) по названию элемента находим функцию в click/change
3) открываем/закрываем соответствующие окна/кнопки, загружаем данные

Таблицы 
1) ссылки на места для вывода таблиц при старте сохраняются в links
2) при построении таблицы месту добавляется имя таблицы name
3) для полей ввода данных сохраняем номера строк и столбцов
4) для кнопок имена функций
5) если выводить нужно объект то пишем имя массива имен полей
manager->имя таблицы -> name_list ->[]
6) указатель на объект вызова внешней команды, пара значений  имя / ссылка
manager->имя таблицы -> link ->ссылка

Команды
1) список команд 
2) из списка команд имена полей
3) если имя соответствует таблице
- массив
- если есть имена полей (в списке полей) то объект

изменение данных в полях
1) если есть data-save=abonent сохраняем в указвнный объект
для этого в объекте links.objects хранятся ссылки на все возможные объекты


Вход и Регистрация
1) заполняю логин пароль и имя (может быть заполнено автоматически)
recovery - нужно делать автоматически
нажимаю регистрация
2) запоминаю логин и имя
получаю код, цвет, название цвета, сессию
3) открываю окно с полем и кодом
4) после подтверждения нажимаю обновить (вто запрос через 3 сек, 3 раза)
