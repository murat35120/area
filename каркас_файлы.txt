Страница состоит из отдельных файлов решающих каждый свою задачу
css
- список переменных  - универсально
- список рисунков, фон  - универсально
- flex форматирование = карткас страницы  - персонально
- style оформление страницы - персонально
  цвет, оформление, flex-basis, padding, margin, медиазапросы  
- анимация, кнопки переходы, открыть закрыть  - универсально

html 
- каркас страницы

js модули в отдельных файлах
- manager - хранение всех используемых для взаимодействия данных, которые не нужно сохранять
  links - ссылки на элементы интерфейса
- click - функции обаботки кнопок
- comm - связь с внешним миром сервер+файлы
- arrs - заготовленные массивы данных 
- answer - функции обработки ответов сервера
- control - все функции
- start - операции перед стартом страницы
создает объекты abonent, manager, temp

при старте, получаем 
- файл настроек оформления
